twopl.gen.R is the code to generate two-parameter logistic (2PL) data.
mg.twopl.mcmc.fast.R is the code to estimate item and person parameters for the multigroup 2PL model.
rmsd.ppmc.fast3.R is the code to compute root mean square deviation (rmsd) statistics using the posterior predictive model checking framework.
multigroup.R is the example code to compute rmsd using ppmc with simulated data.
